# Customer Churn Prediction

## Overview
This project builds a machine learning model to predict customer churn using Python and scikit-learn.

## Skills demonstrated
- Data cleaning and preprocessing
- Feature engineering
- Machine learning model training
- Model evaluation
- Feature importance analysis

## Tools used
- Python
- pandas
- scikit-learn
- matplotlib
- seaborn

## Results
The Random Forest model successfully predicts customer churn with strong accuracy and identifies key drivers of customer attrition.
